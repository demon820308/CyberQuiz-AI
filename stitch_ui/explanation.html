详细解析

Python 是一种广泛使用的高级编程语言，具有以下核心特性：

  1. 1.
    简单易读的语法: Python 的设计哲学强调代码的可读性和简洁性，使用缩进来表示代码块，使得代码更容易于理解和维护。
  2. 2.
    面向对象编程: Python 支持面向对象编程 (OOP)，允许创建类 and 对象，支持封装、继承和多态等面向对象的特性。
  3. 3.
    动态类型: 虽然 Python 是动态类型语言，但这并不属于其“核心特性”。动态类型意味着在运行时确定变量类型，而不是在编译时。
  4. 4.
    编译型语言: Python 是解释型语言，代码在运行时由解释器逐行执行，而不是先编译成机器码。

因此，正确答案是选项 A 和 C。
