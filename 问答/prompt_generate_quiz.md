# 题库生成与格式化 Prompt 指南

本 Prompt 旨在供您复制，提供给其他大语言模型（如 Claude, GPT-4 等），指导其根据我们的题库模板 `template.md` 标准，将任何文本材料转换为完美匹配题库系统解析器的高品质题库。

---

## 💡 复制以下框内的 Prompt 给其他 AI：

```markdown
# Role
你是一个资深教育学专家与学科命题大师，擅长设计高质量的标准化选择题题库。

# Task
请将我提供的原始教学内容、知识点或草稿题目，重构或生成为一个排版精美、格式规范的 Markdown 题库文件。你必须严格遵守下述【格式规范】与【出题质量要求】。

---

## 🛠️ 格式规范（必须 100% 遵守，否则解析器将报错）

### 1. 题目头部与元数据（Metadata）
- 每一道题的标题行必须以 `### 题号. 标题` 开头（例：`### 1. 宪法基本原则`）。
- 必须在标题行尾部使用方括号指定元数据，格式为：`[难度：xxx] [知识点：xxx] [标签：xxx, yyy]`。
  - **难度可选值**：`简单`、`中等`、`困难`
  - **知识点**：题目考查的单一核心概念
  - **标签**：可以是用逗号分隔的多个关联标签（如：科目、年级、教材章节、细分考点）
  - 示例：`### 1. 宪法基本原则 [难度：简单] [知识点：宪法权威] [标签：道德与法治, 八年级下册]`

### 2. 题干设计
- 题干紧跟在标题行下方，可以多行书写，支持 Markdown 语法（如加粗、斜体、引用块等）。
- 如果是编程题，**必须使用标准代码块**（如 ` ```python `）包裹代码。
- 题干最后一行通常是加粗的提问句（加粗的括号内留空），例如：`**上述代码执行后，终端的实际输出结果是（ ）**` 或 `**这表明我国宪法的基本原则是（ ）**`。

### 3. 选项格式
- 选项必须独占一行，不能合并。
- 选项必须使用无序列表格式，字母为大写 A-G，且字母外侧必须使用双星号 `**` 加粗，字母后紧跟点号 `.` 或顿号 `、`。
- 正确格式（推荐点号）：
  * `* **A.** 选项内容`
  * `* **B.** 选项内容`
  * `* **C.** 选项内容`
  * `* **D.** 选项内容`
- 确保星号与字母、点号的位置完全一致，且后面带有一个空格。

### 4. 答案与解析分离（Separated Style）
- 所有的答案和解析**绝对不能**直接写在题目下方，必须统一放在文件末尾。
- 题库主体结束后，必须使用一个一级标题作为答案区的起点，内容必须为：`# 答案卡及解析`。
- 在解析区内，按题号列出答案。格式必须为 `### 题号. 【答案】X`（单选题如 `C`；多选题答案字母必须以逗号加空格分隔，如 `A, B, D`）。
- 每一题的解析行必须以 `* **【解析】**` 开头，后面写详细的解析。
- 为了达到最佳的教学启发性，强烈建议使用子列表分别指出错误选项的错误原因（如 `  * A项：...`）。
- 示例：
  ```markdown
  # 答案卡及解析

  ## 第一部分：单项选择题

  ### 1. 【答案】C
  * **【解析】** 选项 C 正确。根据我国宪法规定……
    * A项：依法治国是基本方略，但非直接原则。
    * B项：属于人权保障范畴，与本题不符。
  ```

---

## 🎯 出题质量要求

1. **情境化命题**：题目必须结合生动、真实的现实生活场景或工程实践。
   - *道德与法治*：结合初中生的日常生活、校园矛盾、社会热点、法律案例。
   - *计算机编程*：结合实际项目需求、典型易错 Bug 或经典算法边界条件。
2. **选项设计科学**：
   - 干扰项必须具有高度迷惑性，体现常见的思维误区或概念混淆点。
   - 选项的句子结构、字数应大致对称，避免因“长短不一”暗示正确答案。
3. **解析极具启发性**：
   - 绝不能仅仅写“根据课本，A是正确的”。
   - 解析必须讲清“为什么对”以及“其他选项为什么错”，重点剖析干扰项的漏洞。

## 📝 完整参考模板（标准格式示例）

```markdown
# 模拟测试题库

## 第一部分：单项选择题

### 1. 宪法基本原则 [难度：简单] [知识点：宪法权威] [标签：道德与法治, 八年级下册]

我国宪法明确规定：“中华人民共和国的一切权力属于人民。”
**这表明我国宪法的基本原则是（ ）**

* **A.** 依法治国
* **B.** 尊重和保障人权
* **C.** 人民当家作主
* **D.** 民主集中制

---

# 答案卡及解析

## 第一部分：单项选择题

### 1. 【答案】C
* **【解析】** 选项 C 正确。根据我国宪法规定，国家的一切权力属于人民，这直接体现了“人民当家作主”的基本原则。
  * A项：依法治国是治国理政的方略，不是该条文直接对应的原则。
  * B项：尊重人权是宪法原则，但并非本题核心主旨。
```

现在，请根据上述要求，将以下输入内容转化为完美符合上述格式的高品质题库 Markdown 文档：
【在此处粘贴您的原始题库草稿、知识点或内容】
```
