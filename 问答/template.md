# 题库模板与编写规范 (CyberQuiz AI Template)

> [!IMPORTANT]
> **核心编写规范：**
> 1. **标题与元数据**：每道题以 `### 题号. 标题 [难度：xxx] [知识点：xxx] [标签：xxx, yyy]` 开头。中括号内的元数据便于系统进行筛选和学习分析。
>    - **难度**可选值：`简单`、`中等`、`困难`
>    - **知识点**：该题考查的核心知识板块
>    - **标签**：可以是用逗号分隔的多个关联标签（如：科目、年级、具体知识细项）
> 2. **选项格式**：选项必须独占一行，格式为 `* **字母.** 选项内容`，字母范围为 A-G，字母后必须有点号 `.` 或顿号 `、`。
> 3. **多行与代码块**：题干和选项中均支持多行文本，题干内支持使用标准 Markdown 代码块（如 ` ```python `）。
> 4. **答案卡与解析**：文件末尾必须包含一个 `# 答案卡及解析` 的大标题，在此大标题下按题号提供对应的答案和解析。答案格式必须为 `### 题号. 【答案】X`（多选题用逗号和空格分隔，如 `A, B, D`）。

# 答案卡及解析

## 第一部分：单项选择题

### 1. 【答案】C
* **【解析】** 选项 C 正确。根据我国宪法规定，我国是人民民主专政的社会主义国家，国家的一切权力属于人民，这直接体现了我国宪法“人民当家作主”的基本原则。
  * A项：依法治国是党领导人民治理国家的基本方略，但不是一切权力属于人民所对应的直接原则。
  * B项：尊重和保障人权是我国的宪法原则，但题干侧重于国家权力归属。
  * D项：民主集中制是我国国家机构的组织和活动原则。

### 2. 【答案】B
* **【解析】** 选项 B 正确。在 Python 中，列表 `list` 属于可变数据类型。
  * 当执行 `y = x` 时，并没有创建列表的副本，而是将 `y` 指向了与 `x` 相同的内存地址。
  * 因此，调用 `y.append(4)` 会直接修改共享的列表对象，使其变为 `[1, 2, 3, 4]`，此时 `x` 的长度即为 4。

## 第二部分：多项选择题

### 3. 【答案】A, B, C
* **【解析】** 选项 A、B、C 正确。社会规则是人们为了维护有秩序的社会环境而共同遵守的程序与规范。
  * 在社会生活中，道德、纪律、法律等是调节人们行为的最主要的社会规则。
  * D项：个人偏好与习惯不具有普遍的约束力和社会契约性，不属于社会规则。

### 4. 【答案】A, B
* **【解析】** 选项 A、B 正确。
  * A项：`git checkout -b` 是最经典的创建并切换到新分支的命令。
  * B项：`git switch -c` 是 Git 2.23 版本后引入的新命令，专门用于创建并切换分支，职责更清晰。
  * C项：该命令仅创建分支，但并不进行分支切换。
  * D项：该命令用于将指定分支合并到当前分支，不属于创建和切换操作。
